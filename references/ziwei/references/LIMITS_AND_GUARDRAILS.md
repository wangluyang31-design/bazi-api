# 边界与限制

## 1. 排盘必须确定

以下项目不能靠猜：

- 命宫 / 身宫
- 宫位地支 / 宫干
- 五行局
- 主星 / 辅星位置
- 四化
- 大限 / 流年

## 2. 解读必须标记为经验推断

推荐显式字段：

- `confidence`
- `ambiguity_warnings`
- `confidence_notes`

## 3. 常见歧义源

- 晚子时按次日还是当日
- 公历 / 农历输入不明确
- 闰月处理
- 时区与真太阳时
- 不同流派对四化表或局法的选择

## 4. 不允许的行为

- 没有盘面事实却给出“你的命宫是……”
- 没有规则表却假设“主星一定这样落宫”
- 把经验性解读写成必然命运
- 只看单宫位就下最终结论

## 5. 推荐的应对方式

当信息不足时，应输出：

1. 当前能确定的部分
2. 当前不能确定的部分
3. 影响差异的关键参数
4. 下一步建议（补 chart_facts、补配置、接入排盘引擎）
